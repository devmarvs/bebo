# Roadmap

## v0.1 (current)
- Core app + custom router
- Middleware (request ID, logging, recovery)
- JSON and HTML rendering with layouts
- Config defaults and env overrides
- Validation helpers
- Examples for API, web, desktop

## v0.2
- Static file serving helper
- Route groups
- Graceful shutdown helper (context cancellation + server shutdown)
- More middleware (CORS, rate limiting, auth skeletons)

## v0.3
- Minimal ORM-like DB helpers (stdlib SQL only)
- OpenAPI scaffolding
- CLI generator

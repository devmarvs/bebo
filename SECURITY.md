# Security Policy

## Supported Versions
The latest v0.x release receives security fixes. Older releases are not maintained.

## Reporting a Vulnerability
- Use GitHub Security Advisories for this repository.
- Include the affected version, reproduction steps, impact, and any suggested fix.
- If advisories are not available, open a minimal issue asking for a private channel and avoid sharing sensitive details publicly.

We will acknowledge reports as quickly as possible and coordinate a fix and release.
